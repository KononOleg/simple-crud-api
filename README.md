# In-memory CRUD API

## How to run the app:

+ clone repo https://github.com/KononOleg/simple-crud-api
+ go to directory `simple-crud-api`
+ ``` npm i```
### Run tests:
```
npm run test
```
### Run app development-mode:
```
npm run start:dev
```
### Run app production-mode:
```
npm run start:prod
```
